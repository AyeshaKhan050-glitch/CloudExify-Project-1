# Introduction

The **Number Guessing Game** is a Java console-based application developed as part of the **CloudExify Summer Internship 2026**. The project is designed to strengthen the understanding of **Object-Oriented Programming (OOP)** concepts by developing an interactive game using Java. It demonstrates the use of classes, objects, constructors, methods, loops, conditional statements, random number generation, exception handling, and file handling.

---

# Project Description

The game begins by asking the player to enter their **name** and **age**. Based on the player's age, the program automatically selects the appropriate difficulty level. A random secret number is then generated according to the selected range, and the player must guess the correct number within a limited number of attempts. After each guess, the game provides hints such as **"Too High"** or **"Too Low"** to help the player reach the correct answer.

To improve the user experience, the application validates all user inputs and accepts only numeric values. If the player enters letters, symbols, or invalid characters, the program displays an appropriate error message without crashing.

* Player Name and Age Registration

* Too High / Too Low Hints

* Score Calculation
* Best Score Tracking
* File Handling for Best Score
* Player Record Management
* Invalid Input Validation

---

# File Handling

The application stores the **Best Score** in a text file and updates it whenever a better score is achieved. It also saves player information such as **Name, Age, Difficulty Level, Attempts, and Score** in a separate file, allowing records to be maintained for future gameplay.
